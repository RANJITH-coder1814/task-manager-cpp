# task-manager-cpp
A console-based Task Management System in C++ that allows users to add, view, mark complete, and delete tasks, with file handling for data persistence.

A simple **console-based Task Management System** built using **C++ and file handling**.

---

## 🚀 Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Data persistence using file handling

---

## 🛠 Technologies Used
- C++
- File Handling
- Data Structures (Vector)
- OOP Concepts
